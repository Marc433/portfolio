Tiris

Tiris is a dark-themed top-down shooter created at Cremaet Studios. It features enemy AI, item pickup and drop systems, Steam achievements, and UI elements like tutorial boxes and intro videos.

Pickup and drop system: DropComponent for entities and InventoryComponent for the player.

Enemy AI: Designed behaviors and components for 3 unique enemy types.

Integrated Steam achievements using Unreal Engine subsystem.

UI work: Tutorial boxes and introductory cutscenes with video integration.

Raftel Engine

A custom C++ graphics engine with OpenGL. Includes OBJ loading, entity system, real-time shadows, HDR, bloom, and PBR. Fully interactive editor via ImGui.

Engine features demonstration: dynamic lighting, physics, and entity picking.

Custom tools and user interface powered by ImGui.

TrankiTanki

Wave-based action game built in Unreal Engine using C++. Players face hordes of enemies while upgrading tanks and managing resources.

Enemy wave system, player upgrades, and dynamic UI elements.

Core gameplay and level design with AI enemy behaviors.
